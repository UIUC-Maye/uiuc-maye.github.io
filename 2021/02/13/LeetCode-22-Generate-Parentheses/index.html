LeetCode 22. Generate Parentheses

1. 题目简述

给出一个数字n，找出所有可能的括号排列形式。

Example 1:

Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
Example 2:

Input: n = 1
Output: ["()"]
